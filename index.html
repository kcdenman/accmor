Use Cases

Smart Agents serve as an interoperability layer connecting large-language-models (LLMs) to Web3 wallets. The LLM uses natural language prompts from the user to approximate the users on-chain intent, which the Smart Agent uses to contruct an on-chain transaction proposal that the end user can review and sign.

By extending LLMs capabilities to on-chain actions, Smart Agents open up a new read-write paradigm between AI agents and blockchain protocols. While the use cases of Smart Agents are limitless, we have curated a list to enable the first generation of Morpheus hackers.
